🐍 Basic Python Questions

This repository contains basic Python programs that help beginners understand fundamental programming concepts such as conditions, loops, strings, and numbers.

📌 Programs Included

1️⃣ Finding the Largest Number

Finds the largest number among given numbers using conditional statements.

2️⃣ Prime Number

Checks whether a given number is a prime number or not.

3️⃣ Sum of Digits

Calculates the sum of digits of a given number using loops.

4️⃣ Vowels Count / Check

Checks or counts the vowels present in a given string.

5️⃣ Largest String

Finds the largest string based on length from given strings.

🧠 Concepts Used

Variables and Data Types

Conditional Statements (if-else)

Loops (for, while)

String Operations

Logical and Comparison Operators

Clone or download the repository

Open terminal / command prompt

Run any program using:
